### Run client

1.export environment

```zsh
source ./scripts/set-env-vars.sh environment/.codef-client.env
```

2. run application

```zsh
cd io-client
yarn
yarn start:dev
```

### Run server

1.export environment

```zsh
source ./scripts/set-env-vars.sh environment/.codef-server.env
```

2. run application

```zsh
cd io-server
yarn
yarn start:dev
```
